Here is a list of tags and emojis you can use for your blog posts. Simply copy and paste the desired tag into the `tag:` variable in your Markdown front matter.

## Core Tags
* **tag: "☕ Personal"** - Great for casual updates, coffee chats, thoughts, or reflections.
* **tag: "🔬 Research"** - For academic posts, papers, and lab updates.

## Future / Expansion Tags
* **tag: "💻 Code"** (or **"💻 Tutorial"**) - For technical deep dives (e.g., Python web scraping, C++, setting up HTCondor).
* **tag: "🎓 Teaching"** - For sharing insights from your time as a TA for Data Science or Discrete Math.
* **tag: "🎤 Events"** - For wrapping up experiences presenting posters (e.g., N+1 Initiative, AFRL).
* **tag: "💡 Ideas"** - For non-academic lightbulb moments or general thoughts.
